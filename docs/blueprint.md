# **App Name**: Andean Canvas

## Core Features:

- Gallery View: Display a grid of Ecuadorian indigenous paintings.
- Detail View: Show individual painting details: image, name, description, and price.
- Navigation: Enable smooth navigation between paintings.

## Style Guidelines:

- Primary color: Deep terracotta (#B35A29), reflecting earth tones.
- Background color: Light tan (#F2E7D3), a desaturated neutral tone for the background.
- Accent color: Muted turquoise (#5D8AA6) to introduce vibrant highlights inspired by indigenous textiles.
- Body and headline font: 'Alegreya', a serif with an elegant, intellectual, contemporary feel; suitable for headlines or body text
- Incorporate indigenous patterns as subtle UI elements and icons.
- Use a clean and spacious layout to showcase the artwork, providing ample breathing room around each piece.
- Subtle transitions when navigating between paintings to enhance user experience.
